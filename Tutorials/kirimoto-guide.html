Learning Kiri:Moto

What Kiri:Moto Is (and Isn’t)

  • ✔ A tool to visualize your 3D model and catch design issues
  • ✔ A way to see how your model is built layer by layer
  • ✘ Not where final slicing settings are chosen

Basic Controls

  • Left-click + drag: Rotate view
  • Scroll wheel: Zoom
  • Right-click + drag: Pan
  • Load: Import your STL file

Using Layer Preview

  • Click the Slice tab at the top of the page to generate layers
  • Use the layer slider on the bottom
  • Watch how the model builds from bottom to top
  • Look for gaps, floating starts, or missing sections

Common Warning Signs

  • First layers printing in mid‑air
  • Very thin vertical walls
  • Long horizontal bridges
  • Very small details
